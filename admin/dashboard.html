Guidelines
  • <strong></strong>: Use the <strong> or <b> tags to make text bold.
  • <em></em>: Use the <em> or <i> tags to make text italic.
  • <u></u>: Use the <u> tag to underline text.
  • <s></s>: Use the <s> or <strike> tags to apply strikethrough to text.
  • <span style="color: red;"></span>: Use the color property to change the color of text.
  • <span style="font-size: 20px;"></span>: Use the font-size property to change the size of text.
  • <span style="font-family: Arial, sans-serif;"></span>: Use the font-family property to change the font family of text.
  • <div style="text-align: center;"></div>
    : Use the text-align property to align text.
  • <span style="text-decoration: underline;"></span>: Use the text-decoration property to add or remove text decorations.
  • <span style="letter-spacing: 2px;"></span>: Use the letter-spacing property to adjust the spacing between characters.
  • Line Break: <br> (place this directly on line)
  • Space between two words (nbsp): &nbsp; (place this directly between two words)
